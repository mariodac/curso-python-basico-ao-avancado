# pip - instalando pacotes e bibliotecas

`pip` é o instalador de pacotes do `python`

Instalar pacotes

```bash
pip install nome_do_pacote
```

Desinstalar pacotes

```bash
pip install nome_do_pacote
```

Mostra pacotes instalados

```bash
pip freeze
```

Exibir versões do pacote disponível

```bash
pip index versions nome_do_pacote
```
